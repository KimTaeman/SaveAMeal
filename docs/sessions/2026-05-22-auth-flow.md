## Decisions Made

- `service_providers.dart` in `lib/services/` rather than in the auth feature, so `firestoreServiceProvider` is reusable by donor/driver/beneficiary features without duplication.
- `_AuthChangeNotifier extends ChangeNotifier` pattern for GoRouter `refreshListenable` — cleaner than making `routerProvider` re-run on auth changes.
- `UserRole` alias: `user_model.dart` imported as `m` in `auth_repository_impl.dart` to avoid clash with domain's `UserRole`.
